# Showroom

Web portal for presenting consultants to customers.

## Development

### Tools

* .NET Core 3.1
* Node.js 13.7 (or later)
* Visual Studio 2019 (or later)

## Instructions

Install the required tools.

Install NPM dependencies:

```
npm install
```

From Client project directory

```
npm link ../../Brand
```

Build the solution:

```
dotnet build
```
